<h2>BMI Calculator</h2>
<h3>Overview</h3>
A BMI calculator is a tool used to compute a person's Body Mass Index (BMI), which is a simple numerical value that helps assess whether someone has a healthy body weight for their height.

This project is built with Node.js and Express. It provides two endpoints — a GET and a POST — to calculate Body Mass Index (BMI) from user-provided height and weight. <br>
The API returns both the BMI value and the corresponding health category
